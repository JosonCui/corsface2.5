2添加用户   /cfMosqueManage/scuser/add.do

   参数：  name	        string       必填
	      loginName     string       必填
	      pwd           String       必填     密码
	      phone         String       必填     电话
	      roleId        int          必填         所属角色Id
	      orgunitId     int          必填     所属组织ID
	      email         String       可选      邮箱
	      memo          String       可选      备注

	  返回结果：

	      pwd           String       必填     密码
	      phone         String       必填     电话
	      roleId        int          必填         所属角色Id
	      orgunitId     int          必填     所属组织ID
	      email         String       可选      邮箱
	      memo          String       可选      备注
